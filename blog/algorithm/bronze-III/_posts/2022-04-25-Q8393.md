## 💡 문제

n이 주어졌을 때, 1부터 n까지 합을 구하는 프로그램을 작성하시오.

## 💡 입력

첫째 줄에 n (1 ≤ n ≤ 10,000)이 주어진다.

## 💡 출력

1부터 n까지 합을 출력한다.

## 💡 예제 입력 1

```
3
```

## 💡 예제 출력 1

```
6
```

## 🚩 문제 풀이

**[주어진 값]**

* 정수 : n

**[구해야 할 값]**

* 1부터 n까지의 합

---

**반복문을 사용**하여 n까지의 합을 구합니다.

## 🚩 소스 코드

``` java
import java.util.Scanner;

public class Main {
 public static void main(String[] args) {  
  Scanner sc = new Scanner(System.in);
  
  int sum = 0;
  int n = sc.nextInt();
  
  for(int i=1; i<=n; i++) {
   sum += i;
  }
  
  System.out.println(sum);
  
  sc.close();
 }
}
```
